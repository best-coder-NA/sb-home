Transfer from Ethereum

If you have Ethereum (ETH) or Ethereum ERC-20 tokens, you can use the Avalanche-Ethereum bridge to bring them to Avalanche and use Pangolin. The bridge allows you to transfer assets between both blockchain networks. The diagram below illustrates how the assets flow from Ethereum to Avalanche to use for Pangolin.

Transferring Ethereum (ETH) to Avalanche

  1. Access the bridge portal here
  2. Select "Wrap Token"
  3. Select "Connect Metamask" and sign into your wallet
  4. Select the "Ethereum Mainnet" network on your MetaMask wallet
  5. Select Ethereum, enter the amount you would like to wrap and then select "Wrap Token". Learn more about what wrapping means here.
  6. Select "Transfer"
  7. Select the Destination Network as "Avalanche"
  8. Select WETH as the token you want to transfer to Avalanche
  9. Enter the amount of WETH you want to transfer
  10. Enter the destination address, which can be found on your MetaMask wallet or you may select the checkbox "I want to send funds to my address". Note that the address starts with "0x".
  11. Select "Start Transfer"

Congratulations! You wrapped your Ethereum and transferred it to Avalanche, which can now be used to swap tokens or add liquidity to a pool on Pangolin.

Transferring Ethereum ERC-20s to Avalanche

  1. Access the bridge portal here
  2. Make sure you are on "Transfer" tab
  3. Select "Connect Metamask" and sign into your wallet
  4. Select "Ethereum Mainnet" network on your MetaMask wallet
  5. Select the Destination Network as "Avalanche"
  6. Select the ERC-20 token that you would like to transfer
  7. Enter the amount of the ERC-20 token
  8. Enter the destination address, which can be found on your MetaMask wallet or you may select the checkbox "I want to send funds to my address". Note that the address starts with "0x".
  9. Select "Start Transfer"

Congratulations! You transferred your ERC-20 token to Avalanche, which can now be used to swap tokens or add liquidity to a pool on Pangolin.
